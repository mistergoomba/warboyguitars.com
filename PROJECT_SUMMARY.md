# WARBOY GUITARS - Custom Guitar Showcase Website

## Project Summary: WARBOY GUITARS E-commerce Showcase

### **Project Overview**

This is a **modern, high-performance e-commerce showcase website** for custom handcrafted guitars, built with **Next.js 15** and featuring **advanced image galleries**, **smooth animations**, and **responsive design**. The project showcases five unique guitar models with **interactive zoom functionality**, **dynamic routing**, and **integrated contact system** for potential customers.

### **Key Technologies Used**

#### **Frontend Technologies**

- **Next.js 15.2.4** - Latest React framework with App Router and Server Components
- **React 19.0.0** - Modern React with hooks and functional components
- **Tailwind CSS 4.0** - Utility-first CSS framework with custom design system
- **Framer Motion 12.18.1** - Advanced animation library for smooth transitions
- **Next.js Image** - Optimized image handling with WebP conversion and responsive sizing
- **Google Fonts** - Custom typography (Black Ops One, Russo One)

#### **Backend Technologies**

- **Next.js API Routes** - Serverless API endpoints for contact form
- **Resend API** - Email service integration for contact form submissions
- **Server Components** - Next.js 15 server-side rendering capabilities

#### **Infrastructure & Deployment**

- **Vercel** - Cloud deployment and hosting platform
- **GitHub** - Version control and CI/CD integration
- **Static Site Generation** - Optimized for performance and SEO

#### **Development Tools**

- **ESLint** - Code linting and quality assurance
- **PostCSS** - CSS processing and optimization
- **Turbopack** - Fast development server (Next.js 15)
